# Profit Performance Analysis by Product Sub-Category (Tableau)

## Overview

This project focused on building an interactive Tableau dashboard to analyze company profit performance across product categories, sub-categories, states, and yearly trends.

The dashboard was designed to help business users quickly understand which products generate the most profit, which areas underperform, and how profitability changes over time.

---

## Business Questions

- Which product sub-categories generate the highest profits?
- Which sub-categories create losses?
- How has profit changed year over year?
- Which states are most and least profitable?
- How do product categories compare overall?

## Dashboard Features

### Profit by Sub-Category

Compare top and bottom performing product groups.

### Annual Profit Trends

Track changes in profitability over time by sub-category.

### Geographic Profit Analysis

Interactive state map highlighting profitable and underperforming markets.

### Dynamic Category Filters

Users can filter the dashboard by category for focused analysis.

---

## Key Insights

- Some sub-categories consistently drive profits while others generate losses  
- Profitability changes significantly across states and regions  
- Certain categories perform stronger over time than others  
- Interactive filters improve decision-making by isolating trends quickly
