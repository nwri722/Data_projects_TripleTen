# TripleTen_projects
Portfolio projects completed during the Business Intelligence Analytics program at TripleTen Bootcamp.


| Project number | Title | Description |
| :-----------: | ----------- |----------- |
| 1 | [E-commerce Cohort Analysis](./E-commerce%20Cohort%20Analysis/)| Built an interactive spreadsheet analysis using raw website event logs to measure user conversion from product views to purchases, create monthly acquisition cohorts, and calculate customer retention trends to support growth strategy decisions. |
| 2 | [Superstore Returns Analysis Dashboard](./Superstore%20Returns%20Analysis%20Dashboard/) | Developed an interactive Tableau dashboard to identify key drivers of product returns by analyzing return rates across customers, products, geography, and time, with actionable recommendations to reduce return volume and improve profitability. |
| 3 | Shopify App Performance Analysis (Power BI) | Built an interactive Power BI dashboard analyzing Shopify App Store data to identify factors influencing app success, including ratings, reviews, developer responsiveness, and market trends using DAX calculations and data modeling. |
| 4 | Venture Capital Investment Database Analysis | Queried and analyzed startup, funding, acquisition, and investor data using SQL to generate insights on venture capital trends, company performance, and investment opportunities for quarterly stakeholder reporting. |
