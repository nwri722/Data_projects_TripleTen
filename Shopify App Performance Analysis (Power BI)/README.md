# Shopify App Performance Analysis (Power BI)

## Overview

This project explores publicly available Shopify App Store data to understand what drives app success. Using Power BI, I built an interactive dashboard to analyze app ratings, review volume, developer responsiveness, and overall marketplace trends.

The goal was to identify patterns that help explain why some apps perform better than others.

---

## Business Questions

- Do apps with more reviews perform better?
- How do ratings relate to popularity?
- Does replying to reviews improve customer sentiment?
- Which developers have the strongest reputations?

---

## Tools Used

- Power BI  
- Power Query  
- DAX  
- Data Modeling  
- Data Visualization  

---

## Dataset

Source: `shopify.xlsx`

Tables included:

- **apps** – App details  
- **reviews** – User ratings and comments  
- **categories** – App categories  
- **apps_categories** – App/category mapping  

---

## Dashboard Sections

### 1. App Landscape

Analyzed the Shopify app marketplace using:

- Total number of apps  
- Review trends over time  
- Relationship between ratings and review count  

### 2. Reviews Analysis

Created custom metrics to evaluate review quality and developer engagement:

- Helpful review score  
- Whether developers replied to reviews  

### 3. Developer Performance

Compared developers based on:

- Total ratings  
- Helpful review averages  
- Responsiveness to customers  

---

## Key Insights

- More reviews do not always mean higher ratings  
- Developer replies are linked to better customer sentiment  
- Some developers consistently outperform others  
- Review quality is more valuable than raw review volume
