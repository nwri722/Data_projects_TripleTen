# Superstore Returns Analysis Dashboard

## Project Overview

This project focused on analyzing product return behavior using Superstore sales data to identify the key drivers of returns and uncover opportunities to reduce operational losses. An interactive dashboard was created to help stakeholders monitor return trends across products, regions, customers, and categories.

## Business Problem

High return rates can negatively impact profitability, logistics costs, and customer satisfaction. The company needed visibility into where returns were happening most often and what factors were contributing to those returns.

## Tools Used

- Tableau
- Excel / CSV Data Preparation
- Dashboard Design
- KPI Reporting
- Data Visualization

## Key Analyses Performed

### Return Trends

- Measured total returned orders
- Calculated return rate percentages
- Compared returned vs non-returned sales

### Category & Product Analysis

- Identified products with highest return volume
- Compared return rates by category and sub-category
- Highlighted problem areas driving losses

### Regional & Customer Analysis

- Analyzed return behavior by state and region
- Identified customers with frequent returns
- Compared market performance

## Dashboard Features

- Interactive filters
- Region and category drilldowns
- Return KPIs
- Trend charts
- Product performance views

## Key Insights

- Certain products and categories generated disproportionately high returns
- Specific regions had elevated return rates
- A small segment of customers drove repeat returns
- Dashboard supports decisions to reduce losses and improve operations
